## Nomenclatura dos arquivos

Arquivos nomeados automaticamento pelo sistema UNIX (Ubuntu 18.04).

Texto + Data + Horário.Formato

Ex.: Captura de tela de *2019-03-29 20-50-14*.png

*Data*: Ano-Mês-Dia Hora-Minuto-Segundo
